# Express Upload Demo avec Multer

Application Node.js et Express permettant le téléversement d’images avec Multer.

## Fonctionnalités

- Upload simple d’image
- Upload multiple
- Upload avec métadonnées
- Stockage des fichiers sur disque
- Validation des types d’images
- Limitation de taille des fichiers
- Gestion des erreurs
- Prévisualisation des images téléversées

---

# Technologies utilisées

- Node.js
- Express.js
- Multer
- HTML5
- CSS3
- JavaScript

---
